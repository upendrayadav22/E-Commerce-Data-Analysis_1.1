# E-Commerce Sales Data Analysis

## Project Overview

This project analyzes an **E-Commerce sales dataset** using Python to understand sales performance, profit trends, and customer segment behavior. The analysis helps identify profitable products, categories, and time-based sales patterns.

## Objectives

* Analyze **monthly profit trends**
* Identify **top-performing product sub-categories**
* Understand **profit distribution by category**
* Compare **sales and profit across customer segments**
* Evaluate **Sales-to-Profit ratio**

## Tools & Technologies

* **Python**
* **Pandas** (Data cleaning and aggregation)
* **Plotly** (Data visualization)
* **Jupyter Notebook**

## Analysis Performed

* Monthly **profit trend analysis**
* **Sales by Sub-Category** analysis
* **Profit by Category** visualization
* **Profit by Sub-Category** comparison
* **Customer Segment sales vs profit analysis**
* **Sales-to-Profit ratio calculation**

## Key Insights

* Some product **sub-categories generate higher sales and profit**
* Certain **months show higher profit trends**
* **Customer segments contribute differently** to sales and profit
* Profit distribution varies across **different product categories**

## Conclusion

This analysis helps businesses understand **sales patterns, profitable product categories, and customer segment performance**, enabling better **business and marketing decisions**.
